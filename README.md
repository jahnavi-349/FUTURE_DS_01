# FUTURE_DS_01 - Sales Data Analysis

## 📌 Objective
Analyze business sales data to identify revenue trends, top-selling products, category performance, and regional insights.

## 📊 Dataset
Sample Superstore dataset containing sales, product, category, and region information.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib

## 📈 Process
- Data Cleaning
- Sales Trend Analysis
- Top Products Identification
- Category-wise Analysis
- Region-wise Performance Analysis

## 🔍 Key Insights
- Sales vary across months
- Top products generate high revenue
- Categories perform differently
- Regions show different sales patterns
